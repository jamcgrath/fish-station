FISH STATION

· Largemouth Bass Series ·

An original monophonic subtractive bass synth — three oscillators into a Moog-style ladder filter, with envelopes, an LFO and an arpeggiator.

Download for macOS
Apple Silicon · VST3 + CLAP · signed & notarized by Apple

Install

  1. Open the downloaded Fish-Station.pkg and follow the installer.
  2. Rescan plug-ins in your DAW (in Ableton: Settings → Plug-Ins).
  3. Add Fish Station to a track as an instrument.

Installs to /Library/Audio/Plug-Ins (VST3 + CLAP). Requires macOS on Apple Silicon. Ableton loads the VST3; hosts with CLAP support can use either.

Inside

3 oscillators — saw/square + sub, detune, hard sync
Ladder filter — 12/24 dB, resonance to self-oscillation
2 envelopes — amp + filter (env / velocity / key-track)
LFO — free or tempo-synced → pitch / filter / amp
Arpeggiator — up / down / up-down, tempo-synced
Resizable UI — 100 / 150 / 200 % zoom
Built from scratch in Rust · not a port or reverse-engineering of any product.
